Get started in 5 minutes

Simple setup, immediate results

1

Install Prerequisites

Make sure you have Node.js 18+ and pnpm installed.

# Install pnpm if you don't have it
npm install -g pnpm
2

Download & Run

Clone the repo and start the app.

git clone https://github.com/brebuilds/reapermadness
cd reapermadness
pnpm install
pnpm dev
3

Enable OSC in REAPER

One-time setup to let the app talk to REAPER.

1. Open REAPER ‚Üí Preferences (Ctrl+P)
2. Go to Control/OSC/Web
3. Click Add ‚Üí OSC
4. Set Local listen port: 8000
5. Click OK
4

Open & Jam!

Go to http://localhost:5173 and start looping!
